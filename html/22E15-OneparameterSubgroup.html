one-parameter subgroup


Let G be a Lie Group. A one-parameter subgroup of G is a group homomorphism

ϕ:ℝ→G

that is also a differentiable map at the same time. We view ℝ additively and G multiplicatively, so that ϕ⁢(r+s)=ϕ⁢(r)⁢ϕ⁢(s).

Examples.

  1. 1.

    If G=GL⁡(n,k), where k=ℝ or ℂ, then any one-parameter subgroup has the form

    ϕ⁢(t)=et⁢A,

    where A=d⁢ϕd⁢t⁢(0) is an n×n matrix over k. The matrix A is just a tangent vector to the Lie group GL⁡(n,k). This property establishes the fact that there is a one-to-one correspondence between one-parameter subgroups and tangent vectors of GL⁡(n,k). The same relationship holds for a general Lie group. The one-to-one correspondence between tangent vectors at the identity (the Lie algebra) and one-parameter subgroups is established via the exponential map instead of the matrix exponential.

  2. 2.

    If G=O⁡(n,ℝ)⊆GL⁡(n,ℝ), the orthogonal group over R, then any one-parameter subgroup has the same form as in the example above, except that A is skew-symmetric: AT=-A.

  3. 3.

    If G=SL⁡(n,ℝ)⊆GL⁡(n,ℝ), the special linear group over R, then any one-parameter subgroup has the same form as in the example above, except that tr⁡(A)=0, where tr is the trace operator.

  4. 4.

    If G=U⁡(n)=O⁡(n,ℂ)⊆GL⁡(n,ℂ), the unitary group over C, then any one-parameter subgroup has the same form as in the example above, except that A is skew-Hermitian (http://planetmath.org/SkewHermitianMatrix): A=-A*=-A¯T and tr⁡(A)=0.
