The Sorgenfrey plane has the following properties:

\begin{itemize}
\item Hausdorff
\item completely regular
\item separable
\end{itemize}

The Sorgenfrey plane does not have the following properties:
\begin{itemize}
\item Lindelof
\item normal
\item second countable
\end{itemize}
